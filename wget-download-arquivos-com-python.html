Fazendo download de arquivos com Python

Wget é uma biblioteca que facilita a implementação de downloads de arquivos com Python. Pode ser usada para diversos fins, web crawling, downloads de websites inteiros de forma recursiva, páginas inteiras de imagens. Ficou famosa por ser mencionada no filme “Social Network”:

First up is Kirkland, they keep everything open and allow indexes on their apache configuration, so a little wget magic is enough to download the entire Kirkland facebook. Kid stuff!

Instalação

$ pip install wget

Exemplo de uso

>>> import wget
>>> url = 'http://www.example.com/caminho/para/arquivo.mp3'
>>> filename = wget.download(url)
100% [................................................] 3841532 / 3841532>
>> filename
'arquivo.mp3'
